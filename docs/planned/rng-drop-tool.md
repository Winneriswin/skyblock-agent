# 计划：SkyBlock 掉落 RNG 工具

> 状态：**占位中** — 页面已预留 `Drop RNG` 视图，逻辑尚未实现。

## 目标

帮助玩家理解 SkyBlock 各类 RNG 掉落（宠物、稀有掉落、附魔等），根据概率估算期望次数，并根据用户实际尝试次数判断「运气」程度。

## 计划功能

### 1. 掉落配置

- 选择或搜索掉落来源（Boss、活动、钓鱼、挖矿等）
- 配置单项或多项掉落及其概率（支持官方数据导入 + 手动覆盖）
- 支持「至少一次命中」vs「全部命中」等不同目标

### 2. 期望次数估计

- 单次尝试成功率 `p` → 首次成功期望次数 `E = 1/p`（几何分布）
- 多项独立掉落：各条期望、联合概率
- 可选：分位数（如「90% 的人会在 N 次内出」）

### 3. 运气评估（用户输入）

用户输入例如：

- 已尝试次数 `n`
- 是否已出 / 出了几次 `k`

输出例如：

- 比期望多/少花了多少次数
- 在同样 `n` 次尝试下，至少出 1 次的概率（若未出 → 有多「黑」）
- 简单标签：正常 / 略欧 / 略非 / 极欧 / 极非（基于标准差或分位，待定）

### 4. 小工具扩展（后续）

- 连抽模拟（Monte Carlo，可选）
- 历史记录保存到 localStorage
- 与 Wiki / NEU 掉落表联动（若数据源可结构化）

## UI 占位（已实现）

- 侧栏导航：**Drop RNG**
- 两个留空卡片：**掉落配置**、**统计结果**

## 技术备注

- 前端即可实现核心数学（几何/二项分布）；后续可考虑 `/api/rng/...` 若需服务端掉落表
- 掉落表数据源待调研：NEU、Wiki、Hypixel 无统一 API
